# Алименты: окончание правила и ежегодный поиск ПМ

- В правило добавлена необязательная дата окончания начисления.
- Месяц окончания включается в расчёт. После него правило не начисляет алименты.
- Проверяется, что дата окончания не раньше даты начала.
- Поиск прожиточного минимума выполняется заново для года из даты начала правила.
- Используются поисковые страницы ГАРАНТа, раздел московских правовых новостей и государственные порталы правовой информации.
- Результаты кэшируются отдельно по каждому году.
- Карточка правила показывает период действия и открывается для редактирования.
