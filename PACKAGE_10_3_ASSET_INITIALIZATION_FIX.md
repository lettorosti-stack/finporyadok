# Пакет 10.3 — исправление добавления имущества

Причина: в базах, созданных до Пакета 10, массив `state.assets` мог отсутствовать. Код Пакета 10.1 выполнял `state.assets.forEach(...)` при загрузке приложения, из-за чего JavaScript останавливался до регистрации обработчика кнопки «+ Добавить имущество».

Исправление:
- `state.assets` и связанные коллекции инициализируются до первого использования;
- сохранена совместимость со старыми базами;
- версия 0.17.3, versionCode 45.
