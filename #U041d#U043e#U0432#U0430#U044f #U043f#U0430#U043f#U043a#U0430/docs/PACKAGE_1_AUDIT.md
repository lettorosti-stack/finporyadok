# Пакет 1 — визуальная и техническая ревизия

## Выполнено

- Подключена утверждённая системная иконка приложения для установки, рабочего стола и списка приложений.
- Созданы launcher-иконки для всех стандартных плотностей Android.
- Удалён внешний фон вокруг плитки; сама плитка и её оформление сохранены.
- Проверены используемые внутренние иконки.
- Смыслово неверные изображения заменены подходящими из финального каталога.
- Сохранены ранее утверждённые размеры и выравнивание главных плиток.
- Проверена основная навигация: все используемые страницы имеют маршруты и возврат на главный экран.
- Создана таблица соответствия «раздел → файл иконки».

## Не входит в пакет 1

- Изменение моделей данных.
- Добавление облачной синхронизации.
- Изменение функционала косметички, ухода, спорта, питания и цикла.
- Замена WebView на Jetpack Compose.
