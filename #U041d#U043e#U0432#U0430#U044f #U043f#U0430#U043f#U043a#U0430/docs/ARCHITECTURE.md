# Целевая архитектура

- UI: Jetpack Compose
- Navigation: Navigation Compose
- Local data: Room + DataStore
- Cloud: Firebase Auth + Firestore + Storage
- AI: собственный backend → OpenAI API
- Images: CameraX / system photo picker
- Health data: Health Connect
- Background tasks: WorkManager
- Dependency injection: Hilt

Модули данных: profile, beautyBag, care, skinDiary, procedures, subscriptions,
devices, cycle, fitness, nutrition, mentalHealth, calendar, analytics, assistant.
