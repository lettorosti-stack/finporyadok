# Пакет 8.1 — выбор категории и семейные лимиты

- Поле категории в форме лимита заменено на список расходных категорий.
- В модель лимита добавлено поле `memberId`.
- Можно выбрать лимит для всей семьи или конкретного члена семьи, включая детский профиль.
- Фактические расходы по персональному лимиту считаются только по операциям выбранного профиля.
- Старые лимиты без `memberId` автоматически считаются семейными.
- Версия: 0.14.1, versionCode 37.
