# Расходы по работе

Добавлено:
- галочка «Расход по работе» в ручной операции и кассовом чеке;
- по умолчанию галочка выключена;
- фильтр «Для возмещения с работы» в операциях;
- сумма и количество рабочих расходов в сводке;
- отдельный блок на главной;
- отметка в карточке операции и деталях;
- колонка в CSV-экспорте;
- значок уведомлений в верхней панели.

Уведомления по страховкам также остаются в интерфейсе приложения и показываются на главной за 7 дней до окончания и после истечения до фактического продления.
