# Package 14.6 — целостность интерфейса

- Все 15 разделов используют централизованную карту иконок.
- Удалены emoji и служебные символы из подписей разделов.
- Быстрый доступ использует те же иконки и перенесён в конец главной страницы.
- Утверждённый порядок главной: счета, платежи, уведомления, категории, последние операции, быстрый доступ.
- Логика базы, миграции и пользовательские данные не изменялись.
- Удалена лишняя вложенная копия проекта с кодированным именем.
