# ВАЖНО: почему прошлые архивы ничего не меняли

В реальном проекте APK собирается из папки `app/`, а не из `android/app/`.

Ранее иконки помещались в неправильный путь:
`android/app/src/main/res/...`

Сборка GitHub Actions их вообще не использовала.

Кроме того, AndroidManifest ссылался на:
`@drawable/ic_launcher`

поэтому файлы `mipmap-*` не могли повлиять на значок.

В этой версии:
- иконки находятся в `app/src/main/res/mipmap-*`;
- AndroidManifest использует `@mipmap/ic_launcher`;
- versionCode повышен с 1 до 2;
- QR запускается нативным сканером Google через камеру;
- результат передаётся обратно в форму чека внутри приложения.

Загрузите ВСЁ содержимое архива в корень репозитория с заменой.
После успешной сборки удалите старое приложение и установите новый APK из нового запуска GitHub Actions.
