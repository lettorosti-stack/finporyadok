# Пакет 6 — облачная синхронизация

Версия 0.12.0, versionCode 33.

Добавлено:
- подключение папки через Android Storage Access Framework;
- поддержка Google Drive, OneDrive и других провайдеров, отображаемых системным выбором папки;
- единый файл `finporyadok-family-sync.json`;
- автоматическая синхронизация при запуске и после сохранения;
- офлайн-работа с последующей отправкой изменений;
- объединение записей по идентификаторам;
- локальный снимок перед применением облачных данных;
- журнал синхронизации и ошибок;
- ручная синхронизация и отключение папки.

Ограничение: синхронизация выполняется при открытом приложении. Фоновая системная синхронизация будет добавляться вместе с системными Android-уведомлениями и WorkManager.
