# Изменяемый период финансовой картины

Добавлен выбор опорного месяца для финансовой картины.

- месяц можно выбрать через календарь;
- кнопки влево/вправо переключают соседние месяцы;
- кнопка «Текущий» возвращает текущий месяц;
- выбранный месяц сохраняется между запусками;
- неделя, квартал, полугодие и год рассчитываются относительно выбранного месяца;
- режим «Всё время» по-прежнему показывает весь массив операций.
