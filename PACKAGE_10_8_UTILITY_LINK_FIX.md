# Пакет 10.8 — исправление привязки ЖКХ к недвижимости

- перечень ЖКХ перечитывается из актуальной базы при каждом открытии формы;
- поддерживаются старые варианты типа `utility` и новые `utilities`;
- коммунальные шаблоны дополнительно распознаются по названию и категории;
- исключены дубли в списке;
- связанные платежи в карточке недвижимости определяются той же логикой.
