# Раздел «Страховки»

Добавлено:
- загрузка PDF страхового полиса;
- стоимость покупки;
- страховая компания и номер полиса;
- даты начала и окончания;
- объект или человек;
- автоматическая привязка к проекту:
  - дом → Дом;
  - автомобиль → Авто;
  - гимнастика → Гимнастика;
  - здоровье → Здоровье;
  - путешествие → Путешествия;
- привязка страховки по гимнастике к ребёнку;
- автоматическое создание расходной операции;
- фильтры по статусу и проекту;
- предупреждение о завершении в течение 45 дней;
- хранение PDF локально в IndexedDB.

Важно: PDF хранится на конкретном устройстве. Метаданные страховки входят в JSON-резервную копию,
но сам PDF-файл в JSON не включается из-за размера. Оригиналы полисов следует дополнительно хранить
в Google Drive или другой внешней папке.
