# План развития приложения

## Текущая версия
Android-приложение отображает локальный интерактивный прототип из `app/src/main/assets/index.html`.
Данные темы сохраняются локально через WebView `localStorage`.

## Следующие этапы
1. Перенести ключевые экраны на нативный Android UI.
2. Добавить локальную базу данных Room.
3. Реализовать аккаунты и синхронизацию.
4. Подключить календарь, напоминания, камеру и распознавание составов.
5. Добавить безопасный серверный шлюз для ИИ-ассистента.
6. Подключить Health Connect и обмен данными с приложением «Моя терапия».

## Безопасность
Секретные ключи нельзя хранить в APK, HTML, репозитории или GitHub Actions без Secrets.
