# Исправление расположения Gradle-файлов

В корне репозитория:
- `build.gradle` — только объявляет Android Gradle Plugin с версией.

В папке `app`:
- `app/build.gradle` — содержит настройки Android-приложения и зависимости.

Загрузите оба файла с сохранением путей. Не помещайте `app/build.gradle` в корень репозитория.
