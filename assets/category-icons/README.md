# Category icons

Файлы названы латиницей и используются в `app.js` через относительный путь:

`./assets/category-icons/<filename>.png`

Основные имена: `groceries`, `home`, `car`, `health`, `children`, `education`, `cafe-restaurants`, `clothing-shoes`, `beauty`, `travel`, `entertainment`, `sport`, `gifts`, `pets`, `phone-internet`, `utilities`, `transport`, `insurance`, `finance`, `books`, `taxes-fees`, `repair`, `garden`, `other`, `subscriptions`, `electronics`, `bank-fees`, `leisure`, `charity`, `work`.
