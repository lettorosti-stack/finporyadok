# Finance Icons

Набор из 39 контурных SVG-иконок для категорий семейного бюджета. Все пиктограммы построены на сетке 24×24, используют `currentColor` и одинаковую толщину линии `1.8`.

## Подключение

Добавьте `finance-icons.svg` в начало страницы (или загрузите его как SVG-спрайт), затем используйте символ:

```html
<svg width="24" height="24" aria-hidden="true">
  <use href="/finance-icons/finance-icons.svg#fi-groceries"></use>
</svg>
```

Цвет задаётся через CSS:

```css
.category-icon { color: #f97316; width: 24px; height: 24px; }
```

`categories.json` содержит локализованные названия, смысловые группы и рекомендуемые цвета. Цвет — вспомогательный признак: смысл категории всегда читается по форме и подписи.

## Размеры

- 16 px: плотные списки, толщина линии 1.8 остаётся читаемой.
- 20–24 px: основной интерфейс.
- 32 px: карточки и выбор категории.
- 40–48 px: используйте на цветной подложке 10–14% непрозрачности.

## Состояния

- default: цвет категории;
- selected: белая иконка на заливке цвета категории;
- disabled: `opacity: .38`;
- hover/focus: подложка цвета категории с прозрачностью 10%.

Идентификатор символа всегда равен `fi-` + `id` из JSON.
