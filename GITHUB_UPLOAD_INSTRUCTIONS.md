# ФинПорядок — пакет для загрузки в GitHub

Репозиторий: `lettorosti-stack/finporyadok`

## Что заменить

Загрузите содержимое этой папки **в корень репозитория**, сохраняя структуру каталогов:

- `index.html` — мобильная навигация и иконки разделов;
- `styles.css` — мобильная адаптация, карточки последних операций и стили иконок;
- `app.js` — подключение иконок категорий;
- `icons/` — иконка приложения и иконки разделов;
- `assets/category-icons/` — 30 иконок категорий с латинскими именами.

Файл `andromoney-data.js` и другие существующие файлы удалять или заменять не нужно.

## Важно

GitHub должен сохранить точные относительные пути:

```text
index.html
styles.css
app.js
icons/app-icon.png
icons/home.png
...
assets/category-icons/groceries.png
assets/category-icons/home.png
...
```

После загрузки обновите страницу приложения без кэша или пересоберите мобильное приложение/APK.

## Что исправлено

- иконки категорий загружаются из локальной папки `assets/category-icons`;
- все имена файлов написаны латиницей;
- для импортированных названий категорий добавлено автоматическое сопоставление;
- неизвестная категория получает иконку `other.png`;
- блок «Последние операции» на телефоне преобразуется в вертикальные карточки и не расширяет экран;
- длинные описания переносятся внутри карточки.
