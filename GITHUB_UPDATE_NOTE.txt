ФинПорядок — пакет обновления для GitHub.

Заменить в репозитории:
- index.html
- app.js
- styles.css
- manifest.webmanifest
- assets/
- icons/
- android/ (если собирается APK)

После обновления:
1. очистить кэш WebView;
2. для APK удалить старую установку;
3. пересобрать приложение.

Проверить:
- отображение логотипа;
- иконки разделов;
- добавление операции;
- QR-чек.
