# Исправление импорта банковских PDF

Теперь каждый PDF не создаёт отдельный счёт по имени файла.

Правила:
- Т-Банк и Тинькофф → существующий счёт «Тинькофф»;
- Сбер → «Сбербанк»;
- Альфа → «Альфа-Банк»;
- ВТБ → «ВТБ»;
- Райффайзен → «Райффайзен»;
- Вайлдберриз Банк → «Вайлдберриз Банк».

До добавления операции приложение:
1. определяет банк по содержимому выписки и источнику;
2. сопоставляет его с уже существующим счётом;
3. проверяет совпадение по дате, сумме, счёту и описанию;
4. добавляет только отсутствующие операции.

Также при запуске исправляются уже созданные PDF-счета и операции
перепривязываются к существующим банковским счетам.
