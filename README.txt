Обновление интерфейса ФинПорядок

Замените в папке проекта три файла:
- index.html
- styles.css
- app.js

Путь проекта:
C:\Users\User\Documents\приложение для семейного бюджета

После замены запустите приложение и нажмите Ctrl+F5.
Импорт PDF, включая многостраничные выписки Т-Банка и Альфа-Банка, сохранён.
Категории, проекты и счета открываются по нажатию и показывают операции с датами, описанием, счётом и суммой.
