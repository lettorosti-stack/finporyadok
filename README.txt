ФинПорядок — мобильная адаптация интерфейса

Что входит:
- index.html — исправленная мобильная вёрстка
- styles.css — адаптивные стили без разъезда по ширине
- app.js — логика приложения
- папка icons — иконки приложения и разделов

Что исправлено:
1. Убрано разъезжание интерфейса по ширине.
2. Навигация переделана под мобильный формат.
3. Блоки на главной выстроены в более понятной логике.
4. Добавлены иконки приложения и иконки разделов.
5. Таблицы оставлены внутри прокручиваемых контейнеров, чтобы не ломать ширину экрана.

Как заменить:
1. Скопируйте содержимое архива в папку приложения.
2. Замените существующие index.html, styles.css, app.js.
3. Добавьте папку icons рядом с этими файлами.
4. Перезапустите приложение или обновите WebView/страницу без кэша.

Если у вас приложение упаковано в мобильную оболочку (WebView / APK), то в проект нужно положить:
- index.html
- styles.css
- app.js
- папку icons
