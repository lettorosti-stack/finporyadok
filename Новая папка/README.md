# Красота внутри и снаружи

Android-прототип персонального приложения для ухода, здоровья, красоты и внутреннего баланса.

## Обновление финального дизайна
- подключён утверждённый пользователем финальный набор из **147 иконок**;
- все иконки приведены к формату **PNG RGBA 512×512**;
- фон удалён, пропорции сохранены, изображения не растянуты;
- старые наборы «Пудровая роза / Шалфей / Тёмный люкс» удалены из приложения;
- текущий интерфейс использует единый премиальный пудрово-розовый стиль;
- полный каталог расположен в `app/src/main/assets/icons/final/catalog`;
- соответствие смыслов и номеров находится в `app/src/main/assets/icons/final/aliases.json`.

## Что уже включено
- главная страница и бьюти-ассистент;
- косметичка и анализ средств;
- уход за кожей, процедуры, спорт, питание и ментальное здоровье;
- календарь и отображение данных цикла;
- локальная работа без сервера;
- GitHub Actions для автоматической проверки проекта и сборки APK.

## Технические данные
- имя приложения: `Красота внутри и снаружи`;
- package/application ID: `ru.lettorosti.beautyinsideout`;
- minSdk: 24;
- targetSdk / compileSdk: 35;
- Java: 17;
- Android Gradle Plugin: 8.7.3;
- Gradle в GitHub Actions: 8.9.

## Сборка APK через GitHub
1. Распакуйте архив.
2. Загрузите **содержимое папки проекта** в корень репозитория `lettorosti-stack/my-beauty`.
3. Обязательно загрузите скрытую папку `.github`.
4. Откройте `Actions` → `Build Android APK`.
5. Запустите `Run workflow` или дождитесь запуска после push в `main`.
6. Скачайте артефакт `krasota-vnutri-i-snaruzhi-debug-apk`.

Внутри артефакта будет `app-debug.apk`.

## Важно
Это интерактивный UI-прототип в Android WebView. Облачная синхронизация, ИИ-поиск, распознавание составов, Health Connect и интеграция с «Моей терапией» пока представлены интерфейсом и подключаются отдельными этапами. Секретные API-ключи нельзя хранить в репозитории, HTML или APK.

Отображаемый размер всех иконок уменьшен на 20% через CSS. Исходные PNG-файлы не изменялись.

Иконки четырёх главных пространств на домашнем экране уменьшены и выровнены по центру плиток. Исходные PNG-файлы не изменялись.


## Полный функциональный прототип

Добавлены рабочие локальные модули: косметичка, рутины и дневник кожи, календарь, задачи, спорт, питание, рецепты, меню, список покупок, вода, сон, настроение, дыхание, измерения, цикл, локальный бьюти-ассистент, экспорт и импорт данных. Данные сохраняются в localStorage внутри WebView.

## Пакет 1: визуальная ревизия

Подключена финальная системная иконка Android и выполнен аудит соответствия внутренних иконок разделам. Подробная карта находится в `docs/ICON_MAPPING.md`, отчёт — в `docs/PACKAGE_1_AUDIT.md`.

## Пакет №2 — Косметичка и уход

В пакет добавлены расширенные карточки косметических средств, контроль сроков и остатка, фильтры и поиск, привязка средств к этапам рутин, редактируемые утренние/вечерние рутины и расширенный дневник состояния кожи. Старые локальные данные автоматически дополняются новыми полями при загрузке.
