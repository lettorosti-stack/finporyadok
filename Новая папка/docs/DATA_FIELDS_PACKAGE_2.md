# Поля данных пакета №2

## Product
`id`, `name`, `brand`, `category`, `purpose`, `zone`, `actives`, `opened`, `expiry`, `pao`, `volume`, `remaining`, `price`, `store`, `routineTime`, `status`, `rating`, `reaction`, `repurchase`, `notes`, `photo`.

## Routine
`id`, `name`, `period`, `days`, `steps`, `done`.

Каждый элемент `steps`: `title`, `productId`.

## SkinEntry
`id`, `date`, `score`, `dryness`, `oiliness`, `sensitivity`, `redness`, `breakouts`, `cycleDay`, `products`, `photo`, `note`.
