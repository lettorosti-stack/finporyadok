# Обновление 0.7.1

## Возмещение
В карточке рабочего расхода поле «К возмещению» заполняется автоматически остатком по операции. Поле «Фактически возмещено» используется только для реально полученной компенсации.

## Алименты от прожиточного минимума
Правило начисления поддерживает:
- регион;
- сумму прожиточного минимума на ребёнка;
- процент от прожиточного минимума;
- автоматический расчёт суммы на одного ребёнка;
- номер постановления;
- ссылку на официальный текст постановления.

Для нового правила по умолчанию выбран регион Москва и 100% прожиточного минимума.
