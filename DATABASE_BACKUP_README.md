# Экспорт и сохранение базы

Реализовано:
- автосохранение после каждого изменения;
- полная резервная копия JSON;
- экспорт всех операций CSV;
- нативное сохранение через системный выбор папки Android;
- восстановление полной базы;
- объединение резервной копии с текущими данными без дублей.

Раздел: Настройки → Резервные копии и экспорт.

Важно: удаление приложения или очистка его данных удаляет локальную базу.
Храните JSON-копию во внешней папке или облаке.
