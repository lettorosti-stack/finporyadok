# Продавец и редактирование позиций чека

- Поле продавца — `receiptMerchant` / `payee`.
- QR кассового чека не содержит название продавца, поэтому приложение запоминает связь номера ФН с продавцом из ранее сохранённого чека.
- При повторном сканировании QR с тем же ФН продавец подставляется автоматически.
- При OCR бумажного чека приложение также пытается извлечь название продавца из верхней части чека.
- В деталях операции с чеком добавлена кнопка редактирования. Можно добавлять, удалять и исправлять позиции, количество, единицу, цену и связь с карточкой товара.
- После сохранения история покупок пересобирается по актуальным позициям без дублей.
