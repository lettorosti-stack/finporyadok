# Алименты: дата начисления и поиск постановления

- Начисление за расчётный месяц создаётся 1-го числа следующего месяца.
- Текущий незавершённый месяц не включается в задолженность.
- В помесячном расчёте показываются расчётный месяц и фактическая дата начисления.
- Ссылки на правовые источники открываются во внешнем браузере Android.
- Удалена неработающая ссылка на внутренний поиск ГАРАНТа.
- Для 2026 года используется прямая публикация ГАРАНТа; для других периодов доступен рабочий поиск в браузере.
