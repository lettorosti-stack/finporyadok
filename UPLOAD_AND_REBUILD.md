# Загрузка и пересборка

1. Загрузите ВСЁ содержимое архива в корень репозитория с заменой.
2. Для веб-версии и PWA используются `manifest.webmanifest` и `assets/app-icons/`.
3. Для APK добавлены новые файлы:
   `android/app/src/main/res/mipmap-*/ic_launcher.png`
   `android/app/src/main/res/mipmap-*/ic_launcher_round.png`
   `android/app/src/main/res/mipmap-anydpi-v26/*.xml`
4. APK нужно пересобрать и установить заново. Android не заменяет иконку уже установленного APK одним обновлением HTML.
5. Перед установкой новой сборки удалите старую версию приложения с телефона.

QR доступен в двух местах:
- `+ Операция` → `Сканировать QR чека`;
- `Импорт` → `Кассовый чек по QR`.

При отсутствии BarcodeDetector приложение не показывает системный `file://` prompt:
оно открывает собственное окно и предлагает вставить строку QR, скопированную через Google Lens.
