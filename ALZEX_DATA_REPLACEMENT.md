# ФинПорядок — замена данных на Alzex

В этой сборке старые транзакции приложения удалены из активного хранилища.
При первом запуске приложение загружает только данные из `alzexUTF.csv`.

## Состав данных
- все строки из CSV: 185;
- расходы, доходы и переводы;
- счета;
- категории и подкатегории;
- контрагенты;
- члены семьи;
- проекты;
- комментарии;
- остатки;
- уникальные идентификаторы Alzex.

## Загрузка в GitHub
Загрузите содержимое этой папки в корень репозитория с заменой файлов.

Особенно важно загрузить:
- `app.js`;
- `andromoney-data.js`;
- `index.html`;
- `styles.css`;
- папки `assets` и `icons`.

После публикации:
1. полностью закройте приложение;
2. очистите данные сайта/старого WebView или удалите старую установленную версию;
3. откройте приложение заново.

Старое хранилище `finporyadok.state.v2` больше не используется.
